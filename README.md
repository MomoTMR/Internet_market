# Hop & Barley - Интернет-магазин ингредиентов для пивоварения

Проект выполнен в рамках учебного курса. Представляет собой полноценный интернет-магазин с Web-интерфейсом и REST API.

## Технологии
- **Backend:** Django 5.2, DRF
- **Database:** PostgreSQL
- **Infrastructure:** Docker, Docker Compose
- **Quality:** Pytest, Flake8, Mypy
- **Auth:** JWT (SimpleJWT)

## Структура проекта
- `config/` — основные настройки проекта, маршрутизация и WSGI/ASGI.
- `products/` — управление каталогом: модели товаров, категорий, поиск и фильтрация.
- `orders/` — корзина (session-based) и логика оформления заказов.
- `users/` — кастомная модель пользователя, регистрация и профили.
- `reviews/` — система отзывов и оценок (модели).
- `static/` — стили (CSS), скрипты (JS) и изображения.
- `templates/` — HTML-шаблоны страниц.
- `scripts/` — утилиты для импорта/экспорта данных и инициализации.

## Запуск проекта
0. Клонируйте репозиторий:
1. Создайте файл `.env` на основе `.env.example`
2. Запустите контейнеры:
   ```bash
   docker-compose up --build
   ```

Создайте суперпользователя:
   ```bash
   docker-compose exec web python manage.py createsuperuser
   ```

## Работа с API (JWT)
Документация Swagger доступна по адресу: `http://127.0.0.1:8000/api/docs/`

**Пример получения доступа:**
1. **Получение токена:**
   `POST /api/token/` с данными `username` и `password`.
2. **Использование:**
   Добавьте заголовок `Authorization: Bearer <your_access_token>` к вашим запросам.
